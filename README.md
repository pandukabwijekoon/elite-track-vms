# EliteTrack Performance 🏎️
### The Ultimate Digital Ecosystem for Luxury & Performance Vehicles

EliteTrack is a high-density MERN stack application engineered for the discerning driver. It provides a surgical-grade environment to track, document, and master your personal vehicle garage.

## 🚀 Key Features
- **My Digital Garage**: A high-fidelity headquarters for your entire fleet.
- **Evolution Chronicles**: Document your build's transformation with side-by-side "Before & After" native image uploads.
- **AI Guardian**: Real-time vehicle health scoring and performance analytics.
- **Maintenance Nexus**: Surgical logging of every maintenance event and part upgrade.
- **EliteBot Assistant**: A persistent AI Mechanic providing live diagnostics and vault alerts.
- **The Gloverbox**: A secure digital vault for insurance, revenue licenses, and emission certificates.

## 🛡️ Security & Deployment
This project is built with a **Security-First** mindset using Environment Variables to protect your data.

### 🛠️ Local Setup
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  **Server Setup**:
    - Build your `.env` file using the `server/.env.example` blueprint.
    - Install dependencies: `cd server && npm install`
    - Start Backend: `npm start`
3.  **Client Setup**:
    - Install dependencies: `cd client && npm install`
    - Build your `.env.local` using the `client/.env.example` blueprint.
    - Start Frontend: `npm run dev`

## 💎 Tech Stack
- **Frontend**: React, Framer Motion, Lucid Icons, Vanilla CSS.
- **Backend**: Node.js, Express, Multer (Local Storage), Node-Cron.
- **Database**: MongoDB Atlas.
- **Branding**: Elite Stealth UI (Dark Mode).
